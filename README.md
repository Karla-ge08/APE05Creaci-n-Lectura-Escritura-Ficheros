## 💻 Tecnologías Utilizadas

Para resolver y cumplir con los requerimientos de esta práctica autónoma y de experimentación (APE), se diseñaron e implementaron soluciones utilizando lógica algorítmica, programación modular y orientada a objetos enfocada en el manejo de persistencia de datos (.txt):

* ☕ **JAVA** (Clases, objetos y flujos de I/O de archivos)

## 📝 Descripción de la Práctica

El proyecto consiste en desarrollar una serie de soluciones académicas estructuradas que demuestren el uso correcto de clases y métodos nativos para la creación, lectura y escritura de archivos de texto plano.

### 📋 Ejercicios Desarrollados y Requerimientos:

| Ejercicio | Componentes Principales | Requerimientos Técnico | Archivo Generado |
| :--- | :--- | :--- | :--- |
| **1. Registro de Estudiantes** | Clases, Objetos, Métodos y Escritura | Almacenar cédula, nombre, carrera y promedio. | `estudiantes.txt` |
| **2. Lectura de Estudiantes** | Lectura, Ciclos, Arreglos y Métodos | Mostrar registros, contar registros y calcular promedio general. | *(Lectura)* |
| **3. Sistema de Inventario** | POO, Archivos y Métodos | Guardar código, cantidad, precio y calcular valor total. | `inventario.txt` |
| **4. Agenda Telefónica** | Archivos, Métodos y Algoritmos de Búsqueda | Operaciones de agregar, buscar y mostrar contactos. | `agenda.txt` |
| **5. Sistema de Notas** | Métodos, POO y Archivos | Guardar 3 notas, calcular promedio y determinar estado (Aprobado/Reprobado). | `calificaciones.txt` |
| **6. Control de Asistencia** | Archivos, Validaciones y Ciclos | Registrar nombre, fecha, estado e historial con conteo de faltas. | `asistencia.txt` |
| **7. Cajero Automático** | POO, Archivos y Métodos | Simulación de transacciones (Consultar, depositar y retirar saldo). | `cajero.txt` |
| **8. Biblioteca Virtual** | POO, Archivos y Métodos | Control de estados de libros (Disponibles / Prestados) y búsquedas. | `biblioteca.txt` |
| **9. Sistema de Ventas** | POO, Métodos y Archivos | Registro de productos, generación de facturas e historial de ventas. | `ventas.txt` |
| **10. Proyecto Integrador** | CRUD Completo, POO, Validaciones y Menú | Menú interactivo con operaciones completas y persistencia total. | `sistema_integrado.txt` |
